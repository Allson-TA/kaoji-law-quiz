# 公務人員考績法修法申論題題庫

為法律顧問講師設計的考績法修法申論題題庫，供學員線上練習使用。

## 內容概要

- **23 題申論題**，分 7 大主題
- 每題含：題目、來源出處、修法時間、資料來源、展開式範答
- 涵蓋範圍：
  - 第一波（已施行）：114/11/19 總統令修正公布第 3、11、12 條
  - 第二波（草案）：115/3/23 銓敘部預告修正草案

## 使用說明

- 每題含「題目」與可展開的「擬答解析」
- 建議學生先自行思考答案，再點擊展開對照範答
- 題目標註難度（普考/高考/地特）與法律狀態（已施行/草案）
